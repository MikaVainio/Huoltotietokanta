# Työkaluja
Seuraavia työkaluja voi käyttää harjoituksessa ja myös omissa projekteissa. Työkaluiksi on valittu avoimen lähdekoodin ohjelmistoja
tai muita ilmaiseksi ladattavia sovelluksia.
