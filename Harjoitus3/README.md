# Dynaamisia sivuja käyttävä Web-palvelin

Harjoituksen palvelin käyttää `Express`-kirjastoa ja näkymien luontiin
`EJS`-kirjastoa.

## Alkutoimet

1. Luo hakemisto sovellusta varten ja anna sille nimeksi `Harjoitus3`.
2. Luo pääohjelman tiedosto `server.js`.
3. Muodosta riippuvuustiedosto `package.json`.
4. Asenna `Express`- ja `EJS`-kirjasto.
5. Luo `public`-hakemisto sivujen elementeille
6. Luo `views`-hakemisto dynaamisia sivuja varten.
